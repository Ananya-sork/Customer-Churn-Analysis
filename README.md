# 📊 Customer Churn Analysis

## Project Overview

This project analyzes customer churn data to identify the key factors influencing customer attrition. The analysis focuses on understanding customer behavior, subscription patterns, and demographic characteristics to uncover insights that help improve customer retention.

---

## Objectives

- Analyze customer demographics and account information.
- Identify churn trends across customer segments.
- Discover factors contributing to customer churn.
- Build an interactive dashboard for business decision-making.

---

## Tools Used

- Microsoft Excel
- SQL
- Python (Pandas, NumPy, Matplotlib)
- Power BI

---

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. SQL-Based Business Queries
5. Data Visualization
6. Interactive Power BI Dashboard

---

## Dashboard Highlights

- Total Customers
- Churn Rate
- Customer Demographics
- Contract Type Analysis
- Monthly Charges Analysis
- Payment Method Insights
- Internet Service Distribution
- Tenure Analysis

---

## Key Insights

- Month-to-month contracts showed the highest churn.
- Customers with higher monthly charges were more likely to leave.
- Electronic check users exhibited a comparatively higher churn rate.
- Longer-tenure customers demonstrated stronger retention.
